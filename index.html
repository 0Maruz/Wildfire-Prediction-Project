What this is

A tree-ensemble regressor predicts days_until_fire (0–7) per 0.1° grid cell across Thailand. Predictions are bucketed into urgency tiers (CRITICAL / HIGH / MEDIUM / LOW) and rendered on a Leaflet map. The held-out test split — the final 20% of data the model never saw during training or selection — is what the accuracy numbers above are measured against.

What the numbers above mean

±1 day accuracy
Share of test-set predictions where the model's predicted fire date fell within ±1 day of the real fire date. Higher = better. 50% means about half the predictions are practically useful.
MAE — mean absolute error (days)
Average distance, in days, between the predicted and the real fire date. Lower = better. MAE of 1.5 means typical predictions are off by 1½ days.
RMSE — root mean squared error (days)
Like MAE but squares errors before averaging, so a few big misses count more than many small ones. Lower = better.
R² — coefficient of determination
How much variance in real fire dates the model explains. 0 = no better than guessing the mean, 1 = perfect, negative = worse than the mean. Wildfire date is noisy, so even R² ≈ 0.3 is useful.
Best model
Either lightgbm or xgboost, picked by lowest validation MAE. RandomForest is included in the candidate pool but consistently loses on this dataset.
Skill check
"Passed" = the model beats the predict-mean baseline by ≥5% on test MAE. "Failed" = the model is barely better than guessing the average — investigate before trusting predictions.
Validation MAE
Used during hyperparameter tuning only. If it's much worse than the test MAE, the validation period was unusually hard (often: off-peak season).
Urgency tiers
Buckets of days_until_fire: CRITICAL = today, HIGH ≤ 2d, MEDIUM ≤ 4d, LOW ≤ 7d. Used by the dashboard for at-a-glance triage.
Predicted cells today
How many 0.1° grid cells the latest run flagged for the next 7 days. Capped at 100 per day so the dashboard stays at operational scale.

Data sources — real only

  • NASA FIRMS VIIRS NRT — satellite hotspot detections; powers fire / FRP / brightness / spatial-neighbour features.
  • Open-Meteo Archive — ECMWF ERA5 daily reanalysis (temp / precip / wind / ET₀), used as features when the cache is present.
  • Hansen GFC v1.11 — baseline canopy density + recent forest loss, distinguishes wildfire vs agricultural-burn signal.
  • GISTDA NRT VIIRS + MODIS — independent hotspot feed from Thailand's national space agency; available as an optional live overlay on the dashboard.
  • Thai province boundaries (CC BY 4.0)

No synthetic, simulated, or interpolated values anywhere in the dashboard. If a real source isn't available, the column is dropped — never faked.

⚠ Disclaimer: This is a research / demo project, not an official emergency-response tool. Predictions are advisory and may not reflect current conditions. For authoritative wildfire information consult GISTDA's official disaster portal or local authorities.
